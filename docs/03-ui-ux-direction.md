# UI and UX Direction

## Visual Theme

Direction: editorial-finance style with warm gradients, glass panels, and high contrast typography.

1. Background: layered gradients with subtle radial highlights.
2. Panels: semi-transparent cards with soft borders and depth.
3. Color system:
   - Primary ink: deep navy
   - Positive: green
   - Negative: red
   - Accent: teal and amber

## Typography

1. Headings: "Space Grotesk" for strong identity.
2. Body and table data: "Manrope" for readability.
3. Numeric emphasis: tabular number styling for amounts.

## Layout Strategy

1. Sticky top bar with title, role switch, and export action.
2. Grid-based dashboard:
   - Summary cards row
   - Two-chart analytical row
   - Transactions and insights row
3. Mobile behavior:
   - Single-column stack
   - Horizontally scrollable table container

## Interaction Patterns

1. Role toggle instantly updates access controls.
2. Filters and search update transaction list in real time.
3. Sort controls are explicit and reversible.
4. Empty and filtered-out states include clear recovery actions.

## Motion Strategy

1. Entrance fade/slide for major sections.
2. Staggered card reveal for summary blocks.
3. Soft hover elevation for actionable elements.

## Accessibility Notes

1. Color contrast targets WCAG AA where practical.
2. Keyboard accessible inputs/buttons.
3. Labels and helper text for all form fields.
4. Charts include textual summaries for non-visual fallback.
